# car-sales-excel-analysis

## 🚗 Car Sales Analysis (Excel)

This project analyzes car sales data using Microsoft Excel to uncover key business insights and trends.

### 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards
- Advanced Formulas

### 📊 Key Analysis
- Total sales by brand and model
- Monthly sales trends
- Top-performing regions
- Revenue analysis

### 📈 Dashboard Features
- KPI Cards (Total Sales, Revenue, Profit)
- Sales trend over time
- Brand-wise performance
- Region-wise sales distribution

### 📌 Key Insights
- Identified top-selling car brands
- Seasonal trends in car sales
- High revenue generating regions
